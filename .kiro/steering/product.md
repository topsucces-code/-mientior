# Product Overview

Mientior is a premium e-commerce marketplace platform built for the French market. It's a full-stack Next.js application featuring a customer-facing storefront and a comprehensive admin panel for managing products, orders, vendors, and marketing campaigns.

**Last Updated:** December 12, 2024

## Core Features

- Multi-vendor marketplace with commission tracking
- Product catalog with variants, images, and reviews
- Shopping cart and multi-step checkout flow
- Payment processing via Paystack and Flutterwave (African payment gateways)
- User accounts with loyalty program (Bronze/Silver/Gold/Platinum tiers)
- Admin dashboard built with Refine.dev and Ant Design
- Marketing tools (campaigns, promo codes, customer segmentation)
- Newsletter subscription system
- Instagram feed integration

## Target Market

- Primary: French-speaking customers (France, West Africa)
- Currency: EUR (Euro)
- Payment methods: Paystack, Flutterwave, with Apple Pay/Google Pay/PayPal support
- Shipping: French addresses with postal code validation

## Business Model

- B2C e-commerce with multi-vendor support
- Vendor commission system (default 15%)
- Loyalty points program for customer retention
- Promotional campaigns via email/SMS/push notifications

## Current Status

The platform is approximately 65% complete with core commerce functionality production-ready. Authentication UI, some admin CRUD operations, and advanced features are still in development.
